# Uccara App

Mobile companion app for the Uccara Sanskrit shloka learning platform. Learn and practice Sanskrit shlokas with interactive word-by-word breakdowns on the go.

## Features

- **Interactive Shloka Learning**: Navigate shlokas line by line, term by term, word by word
- **Pronunciation Guides**: IAST transliteration for accurate Sanskrit chanting
- **Supabase Backend**: Cloud-synced progress and user accounts
- **Cross-Platform**: Built with React Native / Expo for iOS and Android

## Tech Stack

- React Native, Expo
- TypeScript
- Supabase (PostgreSQL + Auth)

## Setup

1. Install dependencies: `npm install`
2. Configure Supabase credentials
3. Run: `npx expo start`
